Privacy Policy

Last updated: July 16, 2026

Context Editor Tab (“the extension”) and the Context Editor web app (“the editor”) are designed so your notes stay on your device. This policy describes what data is involved and what is not.

What the extension does

The Chrome extension’s only job is to open or focus the editor tab when you click its toolbar icon. It requests the tabs permission for that purpose alone. It does not read the content of other websites, and it does not send your notes anywhere.

Where your notes are stored

Notes are saved in your browser’s localStorage for the editor’s origin (for example, https://arverma.github.io). They never leave your browser as part of the app’s design: there is no account system, no sync server, and no analytics backend for note content.

What GitHub Pages serves

The editor’s static HTML, CSS, and JavaScript are hosted on GitHub Pages so the tab uses a normal https:// URL (required for Gemini in Chrome’s tab-context picker). GitHub hosts those static files only; it does not receive or store your note content.

What we do not collect

  • No accounts or sign-in
  • No note content uploaded to any server by this app
  • No advertising or third-party tracking SDKs in the editor
  • No personal information required to use the product

Permissions

The extension uses only the Chrome tabs permission to query for an existing editor tab, focus it, or open a new one at the known editor URL.
